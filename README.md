## Voice Health Assistant (HealthAssistance)

A **voice AI healthcare assistant** designed for medical offices to automate patient outreach and compliance workflows.

### Key Capabilities

**Recording & Compliance**
- Processes call recordings
- Validates against **do-not-call policies**
- Checks **callback timing compliance**
- Stores results and schedules follow-ups

**Voice Agent**
- Built using **LiveKit voice agents**
- Uses:
  - Deepgram (Speech-to-Text)
  - Groq (LLM reasoning)
  - Cartesia (Text-to-Speech)
- Greets patients and interacts conversationally
- Uses database context (patient, recording, job)
- Can confirm or update appointments using tools

**Scheduling**
- Uses **Celery + Redis** for scheduling callbacks
- Supports automated call scheduling

**Browser Voice Testing**
- FastAPI app to generate **LiveKit tokens**
- Allows browser-based voice interaction with the agent
- Useful for local testing and development
